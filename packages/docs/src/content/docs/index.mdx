## API Example

```bash
# Fetch resolved threads
curl http://localhost:3000/api/threads?serverId=123456789&status=resolved
```

```json
{
  "threads": [
    {
      "id": "111222333",
      "title": "How do I implement OAuth?",
      "status": "resolved",
      "messageCount": 8
    }
  ]
}
```
